Página - Teoria das Cores

Nesta pagina encontra-se conteúdo sobre RGB e CMYK, identificando a diferença entre os dois padrões, contém também informações sobre as cores frias, cores quentes e exemplos de paletas.

O Layout utilizado foi de 2 colunas usando flexbox, para separar os temas. Media queries foram necessários, para quando o site for aberto em celular, as colunas ficarem empiladas.

  • Secção 1, header + nav da página.
  • Nas secções A1 e A2, divisão de 2 colunas no layout.
    Foram utilizados 2 imagens com animação de transform.
    Também foi necessário ajustar o texto com 2 media queries.
  • Nas secções B1 e B2, mantém-se a divisão de 2 colunas no layout.
    Os botões se ajustam conforme o tamanho da tela para ficar empilados.
    Na continuação do projeto, após clicar em um dos botões será alterado o plano de fundo e o texto sobre a cor.
  • Secção 2, section com introdução para galeria de fotos
  • Secção 3, galeria de fotos.
    Utilizado um flexbox com 6 imagens, onde são responsivas ao tamanho da tela.
    Foi utilizado uma animação de fadein, das imagens.

Layout das páginas principais

Um formulário onde visitantes podem criar a sua própria paleta.
Foram utlizados os tipos de inputs: color, number, text, radio, select.

Não foi definido nenhum layout especifíco para esta página. Porém foi levado em consideração uma responsividade.

Uma técnica utilizada no input text do código hex, foi deixar em absoluto o simbolo de "#".
Além de considerar o padrão de 6 letras e/ou numeros, com um pattern.
