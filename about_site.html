Página - Teoria das Cores

Nesta pagina encontra-se conteúdo sobre RGB e CMYK, identificando a diferença entre os dois padrões, contém também informações sobre as cores frias, cores quentes e exemplos de paletas.
O Layout utilizado foi de 2 colunas usando flexbos, para separar os temas.


Um media query foi necessário, para quando o site for aberto em celular, as colunas ficarem empiladas.


Página - Adicionar Paleta

Um formulário onde visitante pode criar a sua própria paleta.
Foram utlizados os tipos de inputs: color, number, text, radio, select.
